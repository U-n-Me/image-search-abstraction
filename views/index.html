Image search Abstraction layer

This API service provides links and other image results through get request. This service is towards fulfilment of FCC's back-end development project.

Example usage:

  • https://strong-printer.glitch.me/search/cute%20cat?offset=7
  • https://strong-printer.glitch.me/latest

First example gives a max of 'offset' json results for given query.

Second example gives most recently searched iteams in json format.
